# AI Bitcoin Trader 2

This project aims to create an AI-based Bitcoin trading system that uses various indicators and strategies to make informed trades.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Lemaco82/Ai-bitcoin-trader-2.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Ai-bitcoin-trader-2
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the main application, execute:
```bash
python main.py
```
